## HRDA 雲端智慧面試
HRDA 雲端智慧面試是 HRDA 雲端智慧面試服務在手機上輔助的應用程式.
透過錄影面試，簡化面試流程.不須擔心臨時變動，不用煩惱跨國、遠距人才招聘.
當不方便使用個人電腦或個人電腦瀏覽器不支援使用HRDA服務時,便可使用HRDA錄影面試完成面試流程.
